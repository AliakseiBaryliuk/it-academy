1. Новогодняя елка с различными типами игрушек:
Создадим класс ChristmasTree, который представляет собой новогоднюю елку.
Этот класс будет содержать различные типы игрушек разных цветов.

2. Паттерн мост (Bridge):
Мы можем использовать паттерн мост для разделения абстракции (новогодней елки) от реализации (типа украшений) и
в свою очередь абстракции (тип украшения) от реализации (цвет).
Создадим интерфейс Ornament и реализации этого интерфейса для различных типов украшений.

3. Фабрика (Factory) и Стратегия (Strategy) для украшения елки:
Создадим интерфейс DecorationFactory, который будет представлять фабрику для создания различных типов украшений.
Затем создадим конкретные реализации фабрик для разных стилей украшений (стратегий).
