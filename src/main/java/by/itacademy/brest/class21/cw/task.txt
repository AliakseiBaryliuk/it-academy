2)	Guest содержит информацию о госте, такую как имя, номер комнаты, список процедур, дату въезда и дату выселения.

3)	Room представляет номер в санатории и содержит информацию о своих текущих жильцах (гостях).

4)	Sanatorium содержит список номеров и предоставляет методы для добавления новых гостей, удаления гостей и выполнения процедур.

5)	Реализуйте возможность добавления новых гостей в свободные номера с указанием даты въезда и даты выселения.

6)	Реализуйте метод для удаления гостя по его имени и освобождения номера.

7)	Реализуйте метод для выполнения процедур для всех гостей в санатории.

8) Procedure enum

9) Давайте добавим интерфейс ProcedureProvider, который будет предоставлять метод для выполнения процедур.

10) Реализуйте метод для прохождения процедур гостю в санатории.

11)	Реализуйте метод для вывода информации о текущих жильцах и свободных номерах в санатории.
